# Coria frontend

Next.js app for the Coria team workspace UI.

**Full project docs:** [../README.md](../README.md)

## Local dev

```bash
cp .env.example .env.local
# fill in Supabase + BACKEND_URL + INVOKE_SECRET

npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000). Requires the [backend](../backend) running on port 8000.
